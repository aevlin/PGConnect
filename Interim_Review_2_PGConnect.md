## Introduction
PGConnect is a web-based system designed to simplify PG room discovery and management for students and working professionals.

Key focus:
- Real-time vacancy updates from PG owners
- Comprehensive listing with rent, facilities, amenities, photos, and map location
- Online booking requests
- Owner listing management in real-time
- Centralized interface for users, owners, and admin

---

## Literature Review & Gap Analysis
Existing Platforms
- NoBroker, MagicBricks, 99acres
  - Not PG-specific; general property portals
  - Outdated vacancy information
  - Complex UI; not student-friendly
  - High commission fees
- Co-living platforms (NestAway, Zolo)
  - Limited to major cities
  - Premium pricing
  - Owner has limited self-management

PGConnect Advantage
- Dedicated PG platform for students
- Real-time vacancy updates
- Owner self-management dashboard
- Complete booking workflow
- Interactive map integration
- User reviews and ratings
- Admin verification layer for trust
- Affordable and accessible

## Use Case Diagram (Summary)
Actors and activities:
- Admin: Monitor system activity, verify PG owners, approve listings
- Owner: Upload images, update availability, approve/reject bookings
- User: Search, view details, book rooms, save PGs, add reviews

## Class Diagram (Summary)
Key classes and relationships:
- User, Owner, Admin
- PGListing, Room, Booking, Review, SavedPG
- Relationships: User → Booking/Review/SavedPG; Owner → PGListing; PGListing → Room; Admin → Verification

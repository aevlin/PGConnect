# GitHub Pages Demo

This project includes a static demo version inside `docs/` for GitHub Pages.

## What works on GitHub Pages

- homepage demo
- interactive India map with sample PG markers
- login screen UI
- signup screen UI

## What does not work on GitHub Pages

- PHP
- MySQL
- real login/signup
- backend booking/chat/admin flows

## How to publish

1. Push this repo to GitHub.
2. Open repository Settings.
3. Open Pages.
4. Set source to `Deploy from a branch`.
5. Choose branch `main`.
6. Choose folder `/docs`.
7. Save.

GitHub Pages will then publish `docs/index.html`.
